Conhecimentos sobre o html

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar e estruturar conteúdo na web.Ela é undamental para a criação de páginas web e serve como base para a construção de sites e aplicações web.

O HTML é usado para deinir a estrutura e o layout de uma página web, incluindo elementos como:

  • Títulos e cabeçalhos
    - Parágraos e textos
    - Imagens e vídeos
    - Links e âncoras
    - ormulários e campos de entrada
    - Tabelas e listas

    • O HTML utiliza tags (marcadores) para deinir os dierentes elementos de uma página web. Essas tags são interpretadas pelos navegadores web, que as renderizam como conteúdo visual.
